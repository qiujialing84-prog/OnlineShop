# 代码文件说明
# src/main/java：
controller：
存放 Servlet 类，负责接收前端请求并进行页面跳转或数据处理。
service：
存放业务逻辑代码，用于处理系统的核心功能。
dao：
负责与数据库进行交互，实现数据的增删改查操作。
model：
定义系统中使用的实体类，与数据库表结构对应。

# src/main/webapp
JSP 文件：
用于页面展示。
images 目录：
存放商品图片等静态资源。
WEB-INF/web.xml：
配置 Servlet 映射和 Web 应用的基本运行规则。
